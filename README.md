# SchoolQuiz - AI-Powered Learning Platform

A React.js application that uses AI to create interactive quizzes from uploaded educational materials. Transform vocabulary lists and grammar exercises into engaging learning games!

## Features

### For Hosts (Teachers/Students)
- Upload PDF materials (vocabulary lists, grammar exercises)
- AI automatically generates two types of quizzes:
  1. **Hangman Vocabulary Game**: Interactive word guessing game with Dutch ↔ English translations
  2. **Grammar Exercises**: AI-generated multiple-choice questions based on your materials
- Publish and manage quizzes
- View all your created quizzes

### For Participants
- Join published quizzes
- Play interactive games and earn points
- Compete on the leaderboard
- Track progress across all quizzes

### Game Mechanics

#### Hangman Game
- Guess words correctly to earn up to 100 points (based on speed)
- Each wrong guess draws another part of the hangman
- Win the game by reaching 5,000+ points
- Supports both English-to-Dutch and Dutch-to-English translations

#### Grammar Exercises
- Multiple-choice questions generated by AI
- Instant feedback with explanations
- 100 points per correct answer

## Tech Stack

- **Frontend**: React.js with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Backend**: Supabase (Database, Authentication, Storage)
- **AI**: OpenAI GPT-4 (for quiz generation)
- **Routing**: React Router

## Setup Instructions

### 1. Prerequisites

- Node.js 18+ and npm
- A Supabase account (free tier works)
- An OpenAI API key

### 2. Install Dependencies

```bash
npm install
```

### 3. Set Up Supabase

1. Create a new project at [supabase.com](https://supabase.com)
2. Go to SQL Editor and run the schema from `supabase/schema.sql`
3. Go to Storage and create a bucket named `quiz-materials` with public access
4. Get your project URL and anon key from Settings > API

### 4. Configure Environment Variables

Create a `.env` file in the root directory:

```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_OPENAI_API_KEY=your_openai_api_key
```

### 5. Run the Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Database Schema

The application uses the following main tables:

- `users` - User profiles with roles (host/participant)
- `quizzes` - Quiz metadata
- `hangman_items` - Vocabulary words for hangman games
- `grammar_exercises` - Grammar questions and answers
- `participant_scores` - User scores and statistics
- `quiz_sessions` - Active quiz sessions

## Features in Detail

### Authentication
- Email/password authentication via Supabase Auth
- Password reset functionality
- Role-based access control

### Quiz Creation
- PDF upload to Supabase Storage
- AI processes the PDF and extracts content
- Automatic quiz item generation
- Support for both quiz types

### Game Play
- Real-time scoring
- Progress tracking
- Time-based bonuses (for hangman)
- Immediate feedback

### Leaderboard
- Global ranking by total score
- Shows quizzes completed
- Average score calculation

## Security

- Row Level Security (RLS) enabled on all tables
- Users can only manage their own quizzes
- Published quizzes are viewable by all authenticated users
- Scores are tied to user accounts
